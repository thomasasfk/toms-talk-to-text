## Quite literally a tiny simple python template

---

Setup:

- Install Python 3.x (pyenv recommended)
- Copy `.env.example` to `.env`
```bash
cp .env.example .env
```

- Add x, y, z
```bash
python -m venv .venv
. .venv/bin/activate
python -m pip install -r requirements.txt
```
- Run the script
```bash
python main.py
```
